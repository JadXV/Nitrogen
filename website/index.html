Features

Advanced Code Editor

Monaco powered editor with Roblox based Lua syntax highlighting, auto-completion, and intelligent code suggestions for seamless scripting.

Nitrogen Direct

Execute your saved scripts instantly from the macOS menubar without opening Nitrogen. Quick access to all your scripts with one click - perfect for seamless workflow integration.

ScriptBlox Integration

Browse, search, and execute scripts directly from ScriptBlox with real-time fetching and one-click execution.

Local Script Management

Save, organize, rename, and delete scripts with automatic .lua extension handling and persistent storage.

Auto-Execute System

Toggle auto-execution for scripts to run automatically when joining Roblox games, with Hydrogen autoexec folder integration.

Real-time Console Monitoring

Live Roblox log monitoring with automatic file detection, batch updates, and configurable refresh rates for debugging.

Alt Manager

5 Beautiful Themes
